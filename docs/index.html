OpenLinkHub · Knowledge Base Management

OpenLinkHub RAG

在 LightRAG 检索与生成引擎之上，提供完整的知识库管理平面： 用户与权限、知识库授权、文档操作、流水线监控、流式查询测试与模型配置。

查看源码 架构设计 快速开始

引擎与管理层分离

LightRAG 专注 RAG 能力；OpenLinkHub 负责 RBAC、审计、知识库归属与 API 代理。

开箱即用的管理端

Spring Boot 管理 API + Vue 控制台，覆盖文档上传、索引状态、流式问答与模型配置。

面向三类用户

平台管理员、知识库 Owner、普通使用者 — 权限模型从第一天就纳入设计。

仓库结构

OpenLinkHub-RAG/
  LightRAG/                    # 上游 LightRAG 运行时（独立 clone）
  services/rag-admin-api/      # Spring Boot 管理 API
  apps/rag-admin-web/          # Vue 管理控制台
  docs/                        # 设计文档与 GitHub Pages 站点

运行端口

LightRAG · 127.0.0.1:9621
RAG Admin API · localhost:8091
RAG Admin Web · localhost:5176

快速开始

  1. 在 LightRAG/ 下按上游文档启动 LightRAG 服务。
  2. 复制开发配置： cp services/rag-admin-api/config/application-dev.yml.example services/rag-admin-api/config/application-dev.yml
  3. 启动管理 API：cd services/rag-admin-api && mvn spring-boot:run
  4. 启动 Web 控制台：cd apps/rag-admin-web && npm install && npm run dev
  5. 默认登录：admin / admin123（请在生产环境修改）

当前 MVP 能力
